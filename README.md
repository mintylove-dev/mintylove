# Minty Love site

## Developer pre-requisites
* Install Node.js: https://nodejs.org/
* Install VSCode: https://code.visualstudio.com/
* Install the Svelte VSCode extension: https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode

## Developer setup
* Clone this GitHub repository (this can be done from the "Source Control" tab within VSCode)
* Install dependencies with `npm install`
* Run the dev version of the site with `npm run dev -- --open`

## Deployment
The site will be built & deployed automatically whenever a change is made to the `main` branch of the repository on GitHub
